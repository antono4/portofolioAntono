# portofolioAntono 🎨

![Portfolio Banner](https://images.unsplash.com/photo-1460925895917-afdab827c52f?w=800)

Personal portfolio website showcasing projects and skills.

## ✨ Features

- 👤 Personal introduction
- 💼 Project showcase
- 🛠️ Skills section
- 📬 Contact information
- 📱 Responsive design

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript

## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/antono4/portofolioAntono.git

# Open in browser
cd portofolioAntono
open index.html
```

## 📁 Project Structure

```
portofolioAntono/

├── js/             # JavaScript files
└── assets/         # Static assets
```
